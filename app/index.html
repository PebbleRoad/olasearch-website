Why OlaSearch

  • Simple setup and configuration

    OlaSearch is all front-end so it’s easy to install and set up - no backend plumbing required.

  • Advanced search features

    Start giving your users advanced search features like autosuggest and rich snippets - the same ones you see on sites like Amazon.com

  • Consistent user experience

    OlaSearch can be the single search interface across your organisation, so your users get a unified search experience.

  • Supports many search engines

    OlaSearch works with different search engines such as Solr, Elasticsearch, SharePoint and Swiftype. Different search engine? Contact us.

Features

  • Onboarding

    Not all users are savvy searchers. OlaSearch offers a step-by-step onboarding guide to help users get the most out of the interface.

  • Responsive

    The search interface is responsive. This means you get a smooth search experience on desktops, smartphones and tablets.

  • Autosuggest

    Users can get results as they type, making it quicker to find a known item in a collection.

  • Spellcheck

    OlaSearch supports spelling corrections to help your users so that mistakes don't get in the way of their search.

  • Search filters

    OlaSearch offers search filters that help narrow down a large collection. Choose a checkbox, radio button or slider bar depending on the values.

  • Search history

    OlaSearch keeps track of search keywords, making it easier to re-find an item in a collection.

  • Search snippets

    OlaSearch makes it easy to customise search snippets. These snippets help users to quickly evaluate whether a result is relevant to them.

  • Results bookmarks

    OlaSearch offers search bookmarks, helping users to go back to an item in a collection.

  • Instant search

    See results displayed instantly as you type. OlaSearch is fast and your results are only one keystroke away. Try the demo!

  • Visualisations

    OlaSearch offers different chart types to visualise your collection. You can easily get a bird’s eye view of the collection using aggregates such as count and categories.

  • Themes

    OlaSearch is themeable, so it can follow your company’s look and feel. This makes it easy to introduce OlaSearch as part of your company’s family of apps.

  • Search analytics

    Search work best when you can learn from how people are using it and find opportunities to improve. OlaSearch integrates with Google Analytics, giving you a complete picture of what’s working and what needs to be fixed.
